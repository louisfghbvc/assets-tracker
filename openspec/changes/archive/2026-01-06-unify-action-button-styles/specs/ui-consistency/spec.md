# Capability: UI Consistency

## ADDED Requirements
### Requirement: Unified Action Button Style
The system SHALL use a consistent glassmorphism style for all inline action buttons (Sync, Delete, Refresh).

#### Scenario: Visual Consistency in Assets
- **GIVEN** the user is on the main assets page
- **WHEN** viewing the delete button for an asset
- **THEN** it MUST have a semi-transparent background and glass border, matching the buttons in the Settings tab.
